# 📈 AlphaSignal

> AI-powered Stock Prediction & Decision Support System

## 🚀 Features
- Real-time stock data for Indian (NSE) and US (NASDAQ) markets
- LSTM-based 7-day price prediction
- Buy/Sell/Hold decision engine with explainable AI
- Technical indicators — RSI, MACD, Bollinger Bands, SMA
- News sentiment analysis
- Portfolio tracking
- Risk classification (LOW/MEDIUM/HIGH)

## 🛠 Tech Stack
- **Frontend:** React.js + Tailwind CSS + Recharts
- **Backend:** FastAPI (Python)
- **ML:** LSTM (TensorFlow/Keras)
- **Database:** PostgreSQL (Supabase)
- **Deployment:** Vercel + Render

## 🏃 Run Locally

### Backend
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload

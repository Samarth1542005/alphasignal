## What is AlphaSignal?

AlphaSignal is a placement-ready, production-grade stock analysis platform that lets users search any NSE or NYSE listed stock and instantly get:

- A **BUY / SELL / HOLD** recommendation powered by a weighted technical scoring engine
- A **7-day price forecast** generated by an on-demand LSTM neural network
- **News sentiment analysis** using ProsusAI's FinBERT model
- **Technical indicators** — RSI, MACD, Bollinger Bands, SMA 20/50
- A **personal portfolio tracker** with live P&L — backed by Supabase PostgreSQL

## Features

### LSTM Price Prediction
- Two-layer LSTM neural network trained on 2 years of historical OHLCV data
- 60-day sequence window for pattern recognition
- On-demand training per ticker — works for all 1600+ NSE listed stocks
- Intelligent caching — trained models are saved as `.h5` + `.pkl` files so subsequent requests are instant
- Predicts the next 7 business days with a chart and table view

### AI Decision Engine (BUY / SELL / HOLD)
- Weighted scoring system combining 5 technical indicators
- Returns a confidence percentage (e.g. "BUY — 78% confidence")
- Includes plain-English reasoning (e.g. "RSI is oversold at 28, MACD shows bullish crossover")
- Risk classification — LOW / MEDIUM / HIGH based on volatility

### News Sentiment Analysis
- Fetches latest financial headlines via NewsAPI
- Each headline is scored using ProsusAI's FinBERT — a BERT model fine-tuned on financial text
- Returns per-headline sentiment (positive / negative / neutral) with confidence score
- Overall sentiment breakdown shown as a percentage bar

### Technical Indicators Panel
- RSI (14) — Relative Strength Index with overbought/oversold detection
- MACD — Moving Average Convergence Divergence with signal line and histogram
- Bollinger Bands — Upper, middle, lower bands with position interpretation
- SMA 20 & SMA 50 — Short and medium term trend detection
- Each indicator shown with value, interpretation tag, and plain-English description

### Portfolio Tracker
- Per-user portfolio system backed by Supabase PostgreSQL
- Row Level Security (RLS) — each user can only access their own data
- Add holdings with ticker, quantity, and buy price
- Live P&L calculation using real-time yfinance prices
- Total invested, current value, and profit/loss shown at a glance

### Authentication
- Email + password authentication via Supabase Auth
- bcrypt password hashing and JWT session management
- Email confirmation flow
- Persistent sessions across page refreshes

## Run Locally

### Prerequisites
- Python 3.13
- Node.js 22
- Anaconda (optional)

### Backend
```bash
cd alphasignal/backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Runs on `http://localhost:8000`

### Frontend
```bash
cd alphasignal/frontend
npm install
npm run dev
```

Runs on `http://localhost:5173`

### Environment Variables

Create `backend/.env`:
```
NEWS_API_KEY=your_newsapi_key
```

Create `frontend/.env`:
```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

## ML Model Details

### LSTM Architecture
```
Input: 60-day closing price sequence
       ↓
LSTM Layer 1 (64 units, return_sequences=True)
       ↓
Dropout (0.2)
       ↓
LSTM Layer 2 (64 units)
       ↓
Dropout (0.2)
       ↓
Dense (32, ReLU)
       ↓
Dense (1) → Predicted price
```

- Optimizer: Adam
- Loss: Mean Squared Error
- Early stopping: patience=5
- Training data: 80/20 train/test split on 2 years of daily data

### FinBERT
- Model: `ProsusAI/finbert`
- Task: Financial sentiment classification
- Labels: positive / negative / neutral
- Input: News headline (max 512 tokens)

```sql
  quantity    numeric,

```

Row Level Security enabled — users can only access their own holdings.
